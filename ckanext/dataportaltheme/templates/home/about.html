What Is Data Portal Timişoara

Machine-readable information is data in a format that can be understood by a computer. This means that all relevant fields can be extracted and analyzed without a human contribution. Just because a data set is digitally accessible does not necessarily mean it’s computer readable (more information here). In the spirit of open data, data sets are to be provided in unprotected file formats. Tabular data is provided as CSV files or JSON format. The metadata should be excluded from the data files. These will be included in a zip file with the data file (see the package data point below).

Despre aplicația Data Portal

Scopul platformei Data Portal este de a răspunde la trei probleme legate de datele deschise în România. În primul rând, portalul va deschide publicului larg datele utilizate în cadrul proiectelor Code for Romania. Acestea vor putea fi accesate și vizualizate de către oricine este interesat. În al doilea rând, va defini standarde și protocoale de date pentru seturi de date publicate de către Code4Romania, care vor putea fi implementate de către orice instituție interesată să ofere date publice. În al treilea rând, platforma va oferi un punct de pornire pentru dezvoltarea unei comunități de open data în România. Prin intermediul Data Portal vom oferi sprijin (infrastructură, proceduri, know-how) voluntarilor interesați să deschidă și să curețe data publice din România.

Despre Code for Romania

Upgrading Romania one app at a time. Code for Romania este o organizatie neguvernamentala independenta, neafiliata politic si apolitica. În prezent peste 550 de membri specialisti in IT, comunicare, analiza si design dezvolta solutii digitale pentru a rezolva probleme ale societatii. Construim instrumente software menite sa ajute in cresterea transparentei, a deschiderii datelor publice, sa ajute in sprijinirea educatiei si a accesului la servicii. Ne propunem prin proiectele noastre sa venim in ajutorul tuturor comunitatilor, prin dezvoltarea de aplicatii civice in toate domeniile in care tehnologia poate aduce o schimbare benefica tuturor.
